# Titanic-survival-model
This is my first machine learning model. The purpose of this model is to predicts the passenger's survival rate using binary classification algorithms
## Features
* **Data analysis:** Cleaned messy passenger data, handled missing values, implemented replacing string values to integers.
* **Model evaluation:** decision tree was my first moedel, for the best performance in it I tested multiple max_depth values and concluded with 3 which gave me an 80 accuracy score
* **Optimzation:** Later I optimized the model from decision tree to logistic reggrestion model for a better score of 82.22
## Tools
* **Language:** Python
* **Libraries:** Pandas, Scikit learn / Sklearn
* **Environment:** VS code
